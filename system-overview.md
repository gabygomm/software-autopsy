# Análisis de Sistema

## Repositorio Analizado
https://github.com/astha-gupta9/library-management-system.git

## Descripción General
El proyecto analizado es un sistema desarrollado en Java cuyo objetivo es gestionar
libros, usuarios y préstamos dentro de una biblioteca.

El sistema permite registrar información sobre libros y usuarios y administrar
las operaciones de préstamo y devolución.

## Tecnologías Utilizadas
- Java
- Java Collections Framework
  - ArrayList
  - Map

## Funcionalidades Principales
- Registro de libros
- Registro de usuarios
- Gestión de préstamos
- Devolución de libros

## Estructura General del Proyecto
El proyecto está organizado en varias clases que representan las entidades
principales del sistema, como libros, estudiantes y la gestión de préstamos.

El sistema utiliza estructuras de datos del Java Collections Framework
para almacenar la información en memoria.

## Observaciones Iniciales
En una primera revisión del código se observa que el sistema utiliza
programación orientada a objetos para modelar las entidades principales.

Además, la información parece almacenarse utilizando colecciones de Java
como ArrayList y Map en lugar de utilizar una base de datos externa.

Este documento forma parte de un proyecto de autopsia de software
enfocado en analizar arquitectura, calidad de código y oportunidades
de refactorización.
